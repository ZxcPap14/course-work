# Курсовая работа (4 курс, 2024)
Приложение для управления автосервисом (Windows Forms + SQL Server).

> ⚠️ **Примечание:** Этот проект создан в учебных целях. В нем используются устаревшие подходы (WinForms, прямые SQL-запросы, хранение .bacpac в репозитории). 
> Современная версия этого приложения, написанная на ASP.NET Core 8 + EF Core + PostgreSQL, будет доступна в репозитории [AutoService.Api](ссылка_когда_создадите).
